# Codex image brief — blog hero: "ChatGPT thought I was only a basketball player"

**Open this in the Codex app, generate the image, save output as `chatgpt-baller-hero.webp` in this same folder.**

## What it's for
Hero image for a blog post on yauhenm.com. Post = Yauhen (pro basketball player + AI-search consultant) catching ChatGPT describing him wrong, then fixing it. Image sits at top of an editorial, dark, premium personal site.

## Concept (pick the strongest, ONE image)
A single conceptual image that fuses **basketball** + **AI search** without being literal or cheesy. Best direction:
- A basketball at center, but its black seams morph into a glowing search/chat-bubble outline OR thin circuit/answer lines — like the ball is also a chat answer. Quiet, abstract, premium. No robot, no humanoid AI.
- Alt direction: a split frame — left = textured basketball / court grain, right = a dark "answer box" with faint blurred text lines — same object, two identities. Echoes the post's "same person, two identities."

## Hard style rules (match yauhenm.com brand)
- Background near-black: `#0E0C0A`. Warm dark, not blue-tech.
- One accent: gold `#C9A961` / `#E5C987`. Optional faint rust `#C7593C`. NOTHING else bright.
- Editorial, cinematic, lots of negative space. Looks like a premium magazine cover, not a SaaS blog clipart.
- Texture/grain ok. Soft single light source.
- **NO text, NO words, NO logos** in the image (text added by site).
- **NO** generic "AI" tropes: no glowing brain, no robot, no humanoid, no blue neon grid, no binary rain.
- 16:9, high-res (min 1600×900), export webp.
